IconifyIcon web component makes using icons as easy as icon fonts. To use "tick-circle" in HTML, add this code to the document:

<iconify-icon icon="mdi:tick-circle" style="color: green;" width="50" height="50"></iconify-icon>

Web component will load icon data from Iconify API and render SVG.

Make sure you import IconifyIcon web component:

<script src="https://code.iconify.design/iconify-icon/1.0.7/iconify-icon.min.js"></script>

Web component works with almost all modern UI frameworks. Click documentation link below.
